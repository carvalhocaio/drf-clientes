# DRF School

## Requirements
- [Python](https://www.python.org/downloads/) 3.8.5 or +
- [pipenv](https://pypi.org/project/pipenv/)

## Start Application

**Start virtualenv**  
``` terminal
pipenv shell
```

**Install dependencies**  
``` terminal
pipenv install
```

**Migrations**
``` terminal
python manage.py makemigrations clientes
python manage.py migrate
```

**Run application**  
``` terminal
python manage.py runserver
```
